“Optimizing User Navigation and Cognitive Load in Educational Apps using the Floyd–Warshall Algorithm” is a study that focuses on enhancing the usability, accessibility, and efficiency of educational applications by optimizing how users navigate through different learning modules and system interfaces. The research aims to reduce cognitive load by minimizing unnecessary user interactions, simplifying navigation paths, and improving the organization of educational content. Through the implementation of the Floyd-Warshall Algorithm, the system can analyze all possible navigation routes and determine the shortest and most efficient paths between application pages or features. The study also evaluates how optimized navigation structures can improve user engagement, learning efficiency, task completion speed, and overall user experience in educational platforms. Additionally, the research may incorporate usability testing, behavioral analytics, and interface evaluation techniques to measure the effectiveness of the optimized navigation system.
