# Fortis Nexus — User Acceptance Survey

**Basis:** ISO/IEC 25010 Software Product Quality (Usability) + Domain-Specific Indicators  
**Target Respondents:** Thesis Mentors and Mentee Groups (End Users)  
**Estimated Completion Time:** 10–15 minutes

---

## Instructions

This survey measures your experience using the Fortis Nexus mentor–mentee matching system. Your answers will help determine whether the system meets user needs and improves on the current manual matching process.

Rate each statement based on your actual experience with the system.

| Rating | Meaning |
|--------|---------|
| 5 | Strongly Agree |
| 4 | Agree |
| 3 | Neutral |
| 2 | Disagree |
| 1 | Strongly Disagree |
